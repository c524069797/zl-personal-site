# 欢迎来到我的博客

这是我的第一篇博客文章！这个个人网站使用 Next.js 和 TypeScript 构建，具有以下特点：

## 功能特性

- **博客系统**：支持 Markdown 格式的文章，带有语法高亮
- **简历展示**：清晰的简历页面，支持 PDF 导出
- **主题切换**：支持暗色和亮色主题
- **响应式设计**：在移动设备和桌面设备上都有良好的体验

## 技术栈

- Next.js 16
- TypeScript
- Tailwind CSS
- React Markdown
